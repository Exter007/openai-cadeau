## Setup

1. Make a copy of the example environment variables file

   On Linux systems: 
   ```bash
   $ cp .env.example .env
   ```
   On Windows:
   ```powershell
   $ copy .env.example .env
   ```
2. Add your [API key](https://beta.openai.com/account/api-keys) to the newly created `.env` file

3. Run the app

   ```bash
   $ npm run dev
   ```
